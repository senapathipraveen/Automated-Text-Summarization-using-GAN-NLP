# ✨ Automated Text Summarization using GAN & NLP

This project provides two summarization approaches:

1. Accurate Extractive Summary (NLTK-based)
2. Educational Custom GAN (LSTM-based Generator)

## 🚀 Features
- Streamlit Web Interface
- GAN-based Text Generator
- Frequency-based Extractive Summarizer
- Clean UI with Glassmorphism design
